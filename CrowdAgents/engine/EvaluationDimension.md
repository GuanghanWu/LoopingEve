| firstBlood | 0.20 | 首次击杀新怪物类型 | 低 |

| highFloor | 0.15 | 到达高层（≥8层） | 中 |
| closeVictory | 0.25 | 低血量获胜（HP<20%） | 低 |
| comeback | 0.30 | 低血量翻盘获胜 | 极低 |

| criticalHit | 0.05 | 造成暴击 | 高 |
| lowHPBattle | 0.10/0.15 | 战斗中血量低于30%/15% | 中 |

| battleDrag | -0.08 | 战斗时间过长（>30s） | 中 |

## 正则化规则（能量守恒）

### 基本原则
- 基础分: 0
- 正向最大累积: +10
- 负向最大累积: -10
- 最终范围: 0 ~ 10

### 频率乘数
| 频率 | 乘数 | 说明 |
|------|------|------|
| 极低 | 2.0 | 稀有事件，高分值 |
| 低 | 1.5 | 不常发生 |
| 中 | 1.0 | 正常频率 |
| 高 | 0.7 | 经常发生，分值衰减 |
| 持续 | 0.5 | 持续状态，大幅衰减 |

### 敏感度修正
根据 Agent 类型的敏感度调整分数：
- 正向敏感度: 0.5 ~ 1.5
- 负向敏感度: 0.5 ~ 1.5

### 期望修正
根据 Agent 对维度的期望值调整分数：
- 高期望: 分数提升效果降低
- 低期望: 分数提升效果增强
