# LoopingEve - 循环冒险

一个简单有趣的打怪升级循环小游戏！

## 游戏特色

- 🎮 **简单易上手** - 点击探索遭遇怪物，战斗升级
- ⚔️ **战斗系统** - 普通攻击和强力技能
- 📈 **升级系统** - 击败怪物获得经验值和金币
- 🏪 **商店系统** - 购买装备强化角色
- 🗼 **无限层数** - 挑战更高层数，面对更强敌人

## 如何游玩

1. 点击 **探索** 按钮遭遇怪物
2. 使用 **攻击** 或 **技能** 击败怪物
3. 获得经验值升级，获得金币购买装备
4. 击败3只怪物后可进入 **下一层**
5. 挑战更高层数，成为最强勇者！

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 本地运行

直接在浏览器中打开 `index.html` 即可开始游戏！
